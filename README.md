# proton-test-task

Write a set of automated tests for the following scenarios:

1. Create a label/folder
2. Modify a label/folder
3. Delete a label/folder
4. Change order of labels/folders in the list

## Project info

Web UI automation.
Tools:
Javascript,
Webdriver.io,
Visual Studio Code

## Project Setup

> **Note**

> Check that you have Node v12

`$ npm install`


## To run the test

`$ npm test`
